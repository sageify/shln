Usage:	shln COMMAND

Shell link commands

Commands:
 
Links
 ln       new link
 ls       list links
 mv       rename a link 
 rm       remove a link
 exec     execute link directly
 version  show the git tag, if any, of the source for a link

Script Packages
 tags      list tags for pacakge
 install   install new package
 uninstall uninstall a package
 packs     list packages

Script Modules
 mods      list installed modules

General
 help     show this page
